Restrict woocommerce orders to any specific amount
== Description ==
WooCommerce Order Restriction is a powerful and easy-to-use WordPress plugin designed to help WooCommerce store owners enforce minimum order amounts on their online stores. With this plugin, you can set a minimum order amount, such as $150, and display a customizable alert message to customers who attempt to checkout with an order total below the specified amount. This ensures that your store maintains profitability and encourages customers to increase their order value.

Plugin Documentiation: [https://wordpress.org/plugins/order-restriction/](https://wordpress.org/plugins/order-restriction/)
* [Need help](https://technuxt.com/contact)
* [About Author](https://technuxt.com/)
= Features: =

    * Set Minimum Order Amount: Easily set a minimum order amount in the plugin settings to ensure customers cannot checkout with orders below this threshold.
    * Custom Alert Message: Customize the alert message displayed to customers when their order does not meet the minimum amount requirement.
    * Easy Integration: Seamlessly integrates with WooCommerce, allowing you to manage order restrictions without any complex configuration.
    * User-Friendly Interface: Simple and intuitive admin interface to set up and manage order restrictions effortlessly.
    * Custom Icon: You can use also custom icom for your Message.

= Installation: =

   * Upload Plugin Files: Upload the woocommerce-order-restriction folder to the /wp-content/plugins/ directory.
   * Activate the Plugin: Activate the plugin through the 'Plugins' menu in WordPress.
   * Configure Settings:  Go to  Order Restriction from menu to configure the minimum order amount and customize the alert message.
   * View the notice:Copy the shortcode and use it wherever you need
= Usage: =

    Set Minimum Order Amount:
        * Navigate to Order Restriction from main menu.
        * Enter your desired minimum order amount (e.g., $150).
        * Customize the alert message that will be displayed to customers who do not meet the minimum order requirement.

    Save Changes: Click 'Save changes' to apply your settings.
